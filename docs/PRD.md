## 1. 产品概述

### 1.1 产品定位

魔方简历是一款**开源、免费、隐私优先**的在线简历编辑器，基于 AI 技术辅助用户快速创建专业、精美的简历。核心特点是：

- **无需注册登录**，开箱即用
- **数据完全本地存储**，保护用户隐私
- **AI 智能辅助**，提供润色、语法检查、PDF 识别导入
- **所见即所得**，实时预览编辑效果
- **多种专业模板**，适配不同求职场景

### 1.2 目标用户

- 求职者（校招、社招、转行）
- 自由职业者/独立开发者
- 需要频繁更新简历的职场人士
- 对隐私敏感、不愿将数据上传云端的技术人员

### 1.3 核心价值主张

| 维度 | 价值 |
|------|------|
| **效率** | AI 辅助快速生成专业简历，减少手动编写时间 |
| **隐私** | 数据完全存储在本地，支持本地文件夹同步备份 |
| **专业** | 8 套精心设计的模板，支持深度自定义排版 |
| **灵活** | 支持 JSON/Markdown/PDF 多种格式导入导出 |

### 1.4 产品 Slogan

> "让简历制作变得简单而智能"

### 2.5 导出功能

**导出模态框提供 4 种导出方式：**

| 方式 | 说明 | 技术实现 |
|------|------|----------|
| PDF | 高精度渲染，100% 还原格式，简历投递首选 | 远程 API (`api.magicv.art/generate-pdf`) |
| PDF (备份) | 调用系统打印工具另存为 PDF | 浏览器 `window.print()` |
| JSON 配置 | 保存配置档案副本，支持无损导入恢复 | 本地序列化下载 |
| Markdown | 转换为文本标记语言，方便粘贴到大模型 | HTML 转 Markdown |

**PDF 导出流程：**
1. 克隆预览 DOM 元素
2. 提取并优化样式（过滤字体/动画/悬停等无关样式）
3. 将外部图片转为 Base64
4. 发送 HTML + 样式到远程 PDF 生成服务
5. 下载返回的 PDF 文件

### 2.6 AI 功能

#### 2.6.1 AI 润色

**触发方式：** 富文本编辑器工具栏"AI 润色"按钮

**流程：**
1. 打开润色对话框
2. 可选输入自定义要求（如：请用更技术化的表述、突出量化成果）
3. 点击"开始润色"
4. 流式展示 AI 生成结果
5. 对比原始内容与润色后内容
6. 支持"应用内容"或"重新生成"

**系统 Prompt 约束：**
- 使用更专业的词汇和表达方式
- 突出关键成就和技能
- 保持简洁清晰，使用主动语气
- 严格保留原有 Markdown 格式结构
- 禁止输出前言、说明、总结等额外内容

#### 2.6.2 语法检查

**触发方式：** 底部 Dock "AI 语法纠错"按钮

**流程：**
1. 提取简历所有文本内容
2. 发送给 AI 分析
3. 返回 JSON 格式错误列表
4. 在预览面板高亮错误位置
5. 展示错误详情面板：原文、错误点、建议修改、原因
6. 支持"应用"或"忽略"每个错误

**检查范围：**
- 错别字（如"做为"→"作为"）
- 标点符号错误（如重复标点）
- **不检查**：中英文标点混用、空格使用

#### 2.6.3 PDF 识别导入

**触发方式：** 仪表盘"导入简历"→"导入 PDF"

**流程：**
1. 用户上传 PDF 文件
2. 前端将 PDF 每页转为图片（Canvas 渲染，最多 3 页）
3. 发送图片到 `/api/resume-import`
4. Gemini AI 识别图片内容并输出结构化 JSON
5. 前端解析 JSON 生成简历数据
6. 跳转到工作台编辑

## 3. 数据模型

### 3.1 简历数据结构 (ResumeData)

```typescript
interface ResumeData {
  id: string;                    // UUID
  title: string;                 // 简历标题
  createdAt: string;             // ISO 创建时间
  updatedAt: string;             // ISO 更新时间
  templateId: string;            // 模板 ID
  
  // 基本信息
  basic: {
    name: string;
    title: string;
    employementStatus: string;
    email: string;
    phone: string;
    location: string;
    birthDate: string;
    photo: string;               // 头像 URL/Base64
    photoConfig: PhotoConfig;    // 头像配置
    fieldOrder: BasicFieldType[]; // 字段排序
    icons: Record<string, string>; // 字段图标映射
    customFields: CustomFieldType[]; // 自定义字段
    githubKey: string;
    githubUseName: string;
    githubContributionsVisible: boolean;
    layout: "left" | "center" | "right";
  };
  
  // 教育经历
  education: Education[];
  
  // 工作经验
  experience: Experience[];
  
  // 项目经历
  projects: Project[];
  
  // 证书作品
  certificates: Certificate[];
  
  // 自定义模块数据
  customData: Record<string, CustomItem[]>;
  
  // 专业技能（富文本 HTML）
  skillContent: string;
  
  // 自我评价（富文本 HTML）
  selfEvaluationContent: string;
  
  // 当前激活的编辑模块
  activeSection: string;
  
  // 拖拽中的项目 ID
  draggingProjectId: string | null;
  
  // 菜单模块配置
  menuSections: MenuSection[];
  
  // 全局样式设置
  globalSettings: GlobalSettings;
}
```

## 4. 非功能需求

### 4.1 性能需求

| 指标 | 目标 |
|------|------|
| 首屏加载 | < 3s（ Lighthouse Performance > 80） |
| 编辑响应 | 输入延迟 < 50ms |
| 预览更新 | 内容变更后 < 300ms 刷新预览 |
| PDF 导出 | < 15s（含网络传输） |
| 文件同步 | 防抖 1.5s，避免高频写入 |

### 4.2 兼容性需求

| 平台 | 支持 |
|------|------|
| 浏览器 | Chrome (推荐)、Edge、Firefox、Safari |
| 移动端 | 响应式适配，工作台切换为移动端布局 |
| 文件系统同步 | 支持 File System Access API 的浏览器 |

### 4.3 安全需求

1. **数据隐私**：所有简历数据仅存储在本地 localStorage，不上传服务器
2. **API Key 安全**：用户 AI API Key 存储在本地，通过服务端代理调用
3. **XSS 防护**：富文本内容需转义处理
4. **CORS**：PDF 导出服务支持跨域

### 4.4 可用性需求

1. **离线可用**：核心编辑功能完全离线可用
2. **数据备份**：支持本地文件夹自动同步 + JSON 手动导出
3. **容错**：导入失败时给出明确错误提示
4. **撤销/重做**：富文本编辑器支持操作历史

|------|------|

### 5.2 数据流

```
用户操作 → Zustand Store → localStorage 持久化
                ↓
        文件系统同步 (可选)
                ↓
        简历预览 → TemplateComponent
                ↓
        导出: PDF / 打印 / JSON / Markdown
```

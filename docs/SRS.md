## 2. 总体描述

### 2.1 产品视角

魔方简历是一款基于 Web 的简历编辑器，采用 B/S 架构，核心编辑功能完全在浏览器端运行，数据存储于本地。AI 功能通过服务端 API 路由代理调用第三方大模型服务。

```
┌─────────────┐      ┌─────────────┐      ┌─────────────────┐
│   用户浏览器  │◄────►│  TanStack   │◄────►│   AI 服务商     │
│  (React App) │      │   Start     │      │ (DeepSeek etc.) │
└─────────────┘      └─────────────┘      └─────────────────┘
       │                    │
       ▼                    ▼
┌─────────────┐      ┌─────────────────┐
│ localStorage│      │ api.magicv.art  │
│ (简历数据)   │      │ (PDF 生成服务)   │
└─────────────┘      └─────────────────┘
       │
       ▼
┌─────────────┐
│ 本地文件夹   │
│ (可选同步)   │
└─────────────┘
```

### 2.2 用户类与特征

| 用户类型 | 特征 | 使用场景 |
|----------|------|----------|
| 普通求职者 | 非技术背景，注重易用性 | 选择模板，填写内容，导出 PDF |
| 技术人员 | 注重隐私，习惯自定义 | 深度自定义样式，使用 AI 润色 |
| 频繁更新者 | 需要多版本管理 | 创建多份简历，复制修改 |
| 隐私敏感用户 | 不愿数据上云 | 本地存储 + 本地文件夹同步 |

### 2.3 运行环境

| 环境 | 要求 |
|------|------|
| 浏览器 | Chrome 90+, Edge 90+, Firefox 88+, Safari 14+ |
| 操作系统 | Windows 10+, macOS 10.15+, Linux, Android, iOS |
| 网络 | AI 功能和 PDF 导出需要网络连接；核心编辑功能可离线使用 |
| 存储 | localStorage 可用空间 >= 5MB |

### 2.4 设计与实现约束

1. **隐私优先**：简历数据不得上传至任何服务器（除用户明确授权的 AI API 调用）
2. **无后端数据库**：不使用中心化数据库存储用户数据
3. **开源合规**：依赖库需兼容 Apache 2.0 或更宽松协议
4. **PWA 支持**：需生成 manifest，支持离线缓存

### 4.2 软件接口

#### 4.2.1 AI 润色 API

```
POST /api/polish
Content-Type: application/json

Request:
{
  "apiKey": string,        // AI 服务商 API Key
  "model": string,         // 模型 ID
  "content": string,       // 待润色内容
  "modelType": "doubao" | "deepseek" | "openai" | "gemini",
  "apiEndpoint": string,   // OpenAI 自定义端点（可选）
  "customInstructions": string  // 自定义要求（可选）
}

Response (Stream):
// text/event-stream 流式输出润色后的文本
```

#### 4.2.2 语法检查 API

```
POST /api/grammar
Content-Type: application/json

Request:
{
  "apiKey": string,
  "model": string,
  "content": string,
  "modelType": "doubao" | "deepseek" | "openai" | "gemini",
  "apiEndpoint": string
}

Response:
{
  "choices": [{
    "message": {
      "content": string  // JSON 字符串，包含 errors 数组
    }
  }]
}
```

#### 4.2.3 PDF 识别导入 API

```
POST /api/resume-import
Content-Type: application/json

Request:
{
  "apiKey": string,        // Gemini API Key
  "model": string,         // Gemini 模型 ID
  "images": string[],      // Base64 图片数组
  "locale": "zh" | "en"
}

Response:
{
  "resume": {
    "title": string,
    "basic": { ... },
    "education": [ ... ],
    "experience": [ ... ],
    "projects": [ ... ],
    "skills": [ ... ]
  }
}
```


### 6.2 数据字典

#### 6.2.1 ResumeData

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
